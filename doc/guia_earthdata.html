Como acessar dados do satélite NISAR

Guia completo para criar sua conta gratuita na NASA e autorizar o download de imagens de radar

📋 O que você vai precisar fazer Para baixar dados do NISAR você precisa de uma conta gratuita na NASA (Earthdata) e autorizar o sistema de download da ASF. São apenas dois passos, feitos uma única vez.
1

Criar conta gratuita na NASA Earthdata

Leva menos de 3 minutos · Gratuito para sempre

  1. 1
    Acesse a página de cadastro Clique no botão abaixo para abrir o site da NASA.
  2. 2
    Clique em "Register" (Registrar) Botão azul no canto superior direito da página.
  3. 3
    Preencha o formulário de cadastro Nome de usuário, e-mail e senha. Use um e-mail que você acessa regularmente.
  4. 4
    Confirme seu e-mail A NASA envia um e-mail de confirmação. Clique no link dentro do e-mail. Se não encontrar, verifique a pasta de Spam.
  5. 5
    Faça login no site Com o e-mail confirmado, entre com seu usuário e senha.
🚀 Criar conta gratuita na NASA
⚠️
Atenção: Guarde seu nome de usuário e senha — você vai precisar deles para configurar o programa de download.
Conta criada? Agora faça o segundo passo
2

Autorizar o sistema de download ASF

Feito uma vez, vale para sempre

  1. 1
    Certifique-se de estar logado Entre no site da NASA Earthdata com sua conta antes de continuar.
  2. 2
    Clique no botão abaixo para autorizar o ASF Esse link abre diretamente a página de autorização do sistema de download (Alaska Satellite Facility).
  3. 3
    Clique em "Agree" (Concordo) Uma única confirmação autoriza o acesso aos dados do NISAR.
  4. 4
    Pronto! Agora você pode baixar dados Abra o programa NISAR Downloader (INICIAR.bat), configure suas credenciais e clique em Buscar.
✅ Autorizar o ASF (Alaska Satellite Facility)
⚠️
Importante: O botão acima só funciona se você já estiver logado no site da NASA. Se clicar e aparecer uma tela de login, entre primeiro e depois clique novamente.
3

Configurar o programa NISAR Downloader

Última etapa — feita uma única vez

  1. 1
    Abra o arquivo nisar_config.yaml Está na pasta do programa. Abra com Bloco de Notas ou o RStudio.
  2. 2
    Preencha seu usuário e senha do Earthdata Substitua SEU_USUARIO_EARTHDATA e SUA_SENHA_EARTHDATA pelos seus dados reais.
  3. 3
    Salve o arquivo e feche Não compartilhe esse arquivo — ele contém sua senha.
  4. 4
    Dê duplo clique em INICIAR.bat O navegador abre automaticamente com a interface de download.
✅
Tudo certo! Com os três passos concluídos, clique em "🔍 Buscar produtos" na interface e depois selecione os arquivos para baixar.
❓

Problemas comuns

Soluções rápidas

  1. 1
    Não recebi o e-mail de confirmação da NASA Verifique a pasta de Spam. O remetente é noreply@urs.earthdata.nasa.gov.
  2. 2
    Erro "401 Unauthorized" ao fazer download Você precisa autorizar o ASF (Passo 2 acima). Acesse o link de autorização estando logado no Earthdata.
  3. 3
    Erro "Failed to log in" ou "Invalid credentials" Verifique se usuário e senha no arquivo nisar_config.yaml estão corretos. Tente fazer login manualmente em urs.earthdata.nasa.gov.
  4. 4
    Nenhum produto encontrado na busca Verifique as datas no config — dados disponíveis: 17/out/2025 até 20/jan/2026. Após jun/2026 haverá mais dados calibrados.
